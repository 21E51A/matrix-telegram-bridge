# Matrix Telegram Bridge Project

This project sets up a Matrix Synapse server with Telegram bridge using Docker.

## Features
- Matrix Synapse server
- Telegram bridge (Mautrix)
- Docker deployment
- Secure messaging

## Tech Stack
- Docker
- Matrix Synapse
- Mautrix Telegram
- Linux (AWS EC2)

## Setup
1. Install Docker
2. Clone repo
3. Configure homeserver.yaml
4. Run containers
